# AI Recommendation Logic System

A simple Artificial Intelligence Recommendation System built using Python that recommends items based on user preferences using logic-based similarity matching.

## Project Overview

This project demonstrates the basic concept of recommendation systems by collecting user interests and matching them with available categories using similarity logic.

The system analyzes user preferences and displays personalized recommendations.

## How It Works

### Step 1 — User Input
User enters interests.

Example:
```

technology,movies

```

### Step 2 — Similarity Matching
The system compares user preferences with available categories.

### Step 3 — Score Calculation
Each category receives a similarity score.

### Step 4 — Recommendation Generation
Items are sorted and displayed based on ratings.
